## 🚀 核心案例 3：动态 Header 签名与作用域穿透对抗

### 1. 原理概述
该脚本还原了某招标数据平台（标找找/企查查体系）在发包前动态生成加密请求头 `X-Kzz-Request-Key` 的核心逻辑。
