## 📖 项目简介

本项目摒弃了传统的学院派理论框架，完全以真实 Web 逆向对抗中的**业务场景和核心痛点**为驱动，构建了一套实战知识库。

当你也卡在某个请求头的动态签名，或者是面对满屏乱码的响应体不知所措，亦或是被极验、瑞数等企业级风控拦截，让我们一起从寻找加密参数的底层调试心法起步，一路打穿发包加密、数据解密、动态 Cookie、视觉人机验证，最后直达 JSVMP 与前沿底层协议。

## 🗺️ 主线思路

在真实的高阶 Web 逆向对抗中，切忌一上来就盲目死磕混淆代码。本项目提倡遵循以下“由浅入深、场景降维”的排查主线：

**掌握定位心法** ➡️ **突破发包/收包层标准加密** ➡️ **攻坚企业级动态 Cookie** ➡️ **跨越视觉与人机验证** ➡️ **决战底层协议与黑盒环境**

掌握这条主线，才能在面对成千上万行的风控面条代码时，保持清晰的战略头脑，知道自己处于什么阶段，该用什么工具。

## 📂 核心实战目录讲解

本项目严格按照真实排查场景划分为以下核心模块，方便遇到问题时“按图索骥”：

- **📁 0\_参数定位（心法）**
  - **核心痛点**：不知道加密逻辑在哪。
  - **内容涵盖**：XHR/Fetch 拦截器断点技巧、突破 Promise 异步断层溯源、JSON.parse/Cookie 等高频底层 API 的 Hook 注入脚本。
- **📁 1\_签名参数（请求层加密）**
  - **核心痛点**：请求发不出去，遭遇防篡改与防重放校验。
  - **内容涵盖**：Header 动态哈希签名（如 MD5/SHA 还原）、URL 动态 Query 参数拼接规则、表单与 Payload 核心业务数据加密。
- **📁 2\_数据解密（响应层解密）**
  - **核心痛点**：数据拿到了，但全是 Base64 或乱码。
  - **内容涵盖**：Axios 响应拦截器追踪、标准对称加密（AES/DES）Key 与 IV 的提取、魔改非标位运算的还原。
- **📁 3\_风控 Cookie 合集**
  - **核心痛点**：不带特定 Cookie 直接返回 403 或无限重定向。
  - **内容涵盖**：本地存储防篡改绕过、浏览器环境指纹采集分析、动态高强度 WAF Token 的生成流程破译。
- **📁 4\_滑块与人机验证**
  - **核心痛点**：频繁触发各类验证码卡死自动化流程。
  - **内容涵盖**：滑块轨迹算法模拟（缓动函数与贝塞尔曲线）、乱序背景图还原、点选坐标偏移计算。
- **📁 5_Webpack**
  - **核心痛点**：找到了加密算法，但模块互相依赖，扣不下来。
  - **内容涵盖**：多文件 Chunk 加载器分析、局部函数全局导出技巧、纯净运行环境的剥离。
- **📁 6_Wasm 与 WSS 协议**
  - **核心痛点**：跳出了纯文本 JS 的范畴，进入二进制领域。
  - **内容涵盖**：WebAssembly (.wasm) 逆向浅析与 Python 桥接调用、WebSocket 实时长连接抓包与 Protobuf 解析。
- **📁 7_JSVMP 与黑盒对抗**
  - **核心痛点**：代码控制流彻底平坦化，手动逆向成本极高。
  - **内容涵盖**：纯黑盒调用逻辑、JS 补环境框架基座运行、基于 Python (FastAPI/Flask) 的 RPC 远程接管方案。
- **📁 进阶反混淆（与工具箱）**
  - **核心痛点**：各个实战案例中重复出现的脏活累活。
  - **内容涵盖**：基于 AST (Babel) 的各类解混淆脚本（如去控制流平坦化、解密字符串）、纯净的 Python/JS 标准加解密轮子。

_(注：实战代码中涉及的账号、密码及敏感 Cookie 均通过根目录 `.env` 文件隔离，绝不在云端展示。)_
