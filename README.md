## 📖 项目简介

Web 逆向工程的八大核心模块，构建了一套从底层通信到高层视觉对抗的完整知识体系。本项目不仅解析了网络协议指纹、参数加密定位及密码学算法还原等硬核技术，还深入探讨了前端混淆保护、运行环境检测及人机验证抽离等实战难点。

项目的核心目的在于指导开发者如何突破动态 Cookie、代码混淆以及自定义字体反爬等复杂防御机制。通过揭示数据加密、轨迹伪造与协议逆向的底层逻辑，本资源库旨在为应对现代互联网高强度风控提供系统性的技术与战术指南。


## 🧠 八大核心知识体系 (Modules)

本项目严格按照 Web 逆向对抗的层级深度，划分为以下八大核心模块：

### 📁 01_网络协议与发包底层对抗
* **核心内容**：HTTP/HTTPS 协议底层机制、TLS 握手过程与 JA3 指纹检测原理。
* **解决痛点**：突破基于网络通道层的风控拦截，解决纯 Python 脚本无法模拟真实浏览器底层指纹的问题。

### 📁 02_加密参数定位体系
* **核心内容**：Header 签名、URL参数、Payload 数据、Response 密文以及动态 Cookie 的拦截与追踪。
* **解决痛点**：掌握 XHR/Fetch 断点、事件监听器追溯以及利用油猴 (Tampermonkey) Hook 核心 API（如 `JSON.parse`, `document.cookie`）的寻根技巧。

### 📁 03_算法与密码学破解
* **核心内容**：标准编码（Base64）、散列哈希（MD5/SHA）、对称与非对称加密（AES/DES/RSA）的特征识别与算法还原。
* **解决痛点**：剥离业务逻辑，将前端 JS 的密码学运算完美转化为 Python 或本地 Node.js 可执行的代码。

### 📁 04_前端架构与代码混淆
* **核心内容**：Webpack 模块化打包原理、OB 混淆 (Obfuscator) 分析、AST (抽象语法树) 清洗与 JSVMP (虚拟机保护) 对抗。
* **解决痛点**：应对“能找到入口但扣不出代码”的窘境，实现代码的自动化解混淆与高阶模块导出。

### 📁 05_运行环境与防调试对抗
* **核心内容**：无限 Debugger 绕过、控制台检测突破，以及基于 JSDOM 或 Proxy 的深度浏览器环境补全（指纹伪造）。
* **解决痛点**：解决目标代码在本地 Node.js 运行时报 `window/document is not defined` 的环境缺失问题。

### 📁 06_人机交互验证突破
* **核心内容**：滑块、点选、文字识别等主流验证码底层的轨迹校验逻辑与生物特征行为伪造。
* **解决痛点**：通过算法生成拟真轨迹数组，剥离物理操作，实现自动化的验证闭环。

### 📁 07_视觉渲染欺骗与反爬
* **核心内容**：自定义字体文件 (WOFF/TTF) 映射解析、CSS 坐标偏移还原。
* **解决痛点**：突破“所见非所得”的视觉障眼法，提取出被掩盖的真实数字与文本。

### 📁 08_高级通信与泛 Web 生态
* **核心内容**：WebSocket 长连接通信逆向、Protobuf 二进制流解析，以及基础的微信小程序解包与 JSBridge 接口分析。
* **解决痛点**：跳出传统 HTTP 文本交互的局限，掌握流式数据与跨端泛 Web 环境的数据抓取。
