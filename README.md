## 📖 项目简介

Web 逆向工程的八大核心模块，构建了一套从底层通信到高层视觉对抗的完整知识体系。本项目不仅解析了网络协议指纹、参数加密定位及密码学算法还原等硬核技术，还深入探讨了前端混淆保护、运行环境检测及人机验证抽离等实战难点。

项目的核心目的在于指导开发者如何突破动态 Cookie、代码混淆以及自定义字体反爬等复杂防御机制。通过揭示数据加密、轨迹伪造与协议逆向的底层逻辑，本资源库旨在为应对现代互联网高强度风控提供系统性的技术与战术指南。


## 🗺️ 新手通关路径 (Learning Path)

在真实的高阶 Web 逆向对抗中，切忌盲目死磕代码。本项目提倡遵循以下“由浅入深、降维打击”的排查主线：

**找参数** ➡️ **破标准算法** ➡️ **破混淆代码** ➡️ **破魔改算法** ➡️ **补环境** ➡️ **最终突破企业级 Cookie**

掌握这条主线，才能在面对成千上万行的风控面条代码时，保持清晰的战略头脑。

## 🧠 核心知识体系 (Modules)

本项目严格按照 Web 逆向对抗的真实难度阶梯，将核心知识划分为四大阶段、八大模块：

### 🧱 阶段一：基础追踪与标准对抗（基石篇）
解决“参数在哪”以及“基础加密怎么破”的问题。

* **📁 01_网络通信与发包底层对抗**
  * **核心内容**：HTTP/HTTPS 底层机制、TLS 握手、JA3 指纹检测原理。
  * **解决痛点**：突破基于网络通道层的风控拦截，解决纯 Python 脚本无法模拟真实浏览器底层指纹的问题。
* **📁 02_参数追踪与标准密码学**
  * **核心内容**：Headers/URL/Body 加密参数的 XHR/Fetch 断点拦截、Hook 定位，以及 MD5/SHA/AES/RSA 等标准算法的还原。
  * **解决痛点**：掌握顺藤摸瓜找代码的技巧，能够剥离业务逻辑，将前端未混淆的密码学运算完美转化为后端代码。

### ⚔️ 阶段二：代码架构与黑盒对抗（硬核篇）
解决“代码看不懂”和“算法被魔改”的问题。

* **📁 03_前端架构与AST代码解混淆**
  * **核心内容**：Webpack 模块化打包机制、OB 混淆 (Obfuscator) 分析、AST (抽象语法树) 静态清洗。
  * **解决痛点**：应对“能找到入口但扣不出代码”的窘境，实现恶心代码的自动化解混淆与重构。
* **📁 04_魔改算法与黑盒调用对抗**
  * **核心内容**：识别魔改密码学常数、JSVMP (虚拟机保护) 对抗，以及基于沙盒的 RPC (远程调用) 导出技术。
  * **解决痛点**：当算法被改得面目全非或控制流彻底平坦化时，放弃手动扣代码，利用纯净 JS 沙盒实现黑盒运行与算法接管。

### 🌊 阶段三：环境伪造与高阶风控（深水区）
迎战逆向领域的最终 Boss：环境检测与动态指纹。

* **📁 05_运行环境补充与浏览器指纹**
  * **核心内容**：BOM/DOM 对象补全、Proxy 代理探针拦截、Canvas/WebGL/WebRTC 硬件指纹欺骗。
  * **解决痛点**：解决目标代码在本地 Node.js 运行时报 `window/document is not defined` 的报错，彻底抹平本地与真实浏览器的差异。
* **📁 06_企业级风控与动态Cookie突破**
  * **核心内容**：瑞数、Akamai、加速乐等顶级 WAF 的防护机制分析，应对多代动态脚本生成的高强度指纹密文。
  * **解决痛点**：综合运用 AST 解混淆与数千个环境对象的深度伪造，实现“请求发出前”的动态防御 Token 生成闭环。

### 🚀 阶段四：非标对抗与泛端生态（拓展篇）
跳出纯粹的 JS 逻辑，解决视觉欺骗与跨端问题。

* **📁 07_人机验证与视觉渲染欺骗**
  * **核心内容**：滑块/点选轨迹算法伪造，自定义字体 (WOFF/TTF) 映射解析，CSS 坐标偏移还原。
  * **解决痛点**：剥离物理操作实现验证码的生物特征模拟，突破“所见非所得”的视觉数据掩盖。
* **📁 08_高级通信与跨端泛Web生态**
  * **核心内容**：WebSocket 长连接通信逆向、Protobuf 二进制流解析、微信小程序解包与 JSBridge 分析。
  * **解决痛点**：跳出传统 HTTP 文本交互的局限，掌握流式数据与泛 Web 环境的数据抓取。
